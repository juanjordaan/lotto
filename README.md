## Turfsport Lottery API
#ts-lottery-api

### Requirements
Java 8
Gradle 3 or 4

#### Compile and Test ( with a Test Profile )
gradle test -i




Alternatively:
#### Dev Profile
create application-dev.properties by copying application-test.properties, change database settings and run
gradle bootRun


#### Compile and Package
gradle clean bootRepackage
